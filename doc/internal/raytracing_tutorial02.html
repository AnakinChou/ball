Creating a molecular object

We start with creating a molecular scene.
We create a molecular object by building a polypeptide from an aminoacid sequence.
To do so, open the PeptideBuilder dialog by clicking the menu entry
"Build->BuildPeptide".
Then enter the sequence ARND with the keyboard in the label field at the top. (As an alternative you can also press the upper four amino acid buttons.) This will create a polypeptide with the aminosequence (Alanin, Arginin, Asparagine, Aspartic acid).

Then press the "Build" button.

This will enable the next step of the tutorial.
Press Next to continue.
