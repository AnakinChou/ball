QuEasyViz 1.0 Documentation

- QuEasy Vizualizer -

This programs allows to quickly and easily create regression or classification pipelines.

In order to create a pipeline, simply follow the steps listed below.
Note that no data will be processed until you choose to execute your pipeline.

Steps for the creation of a QSAR pipeline

  1. read input data
  2. create a regression or classification model
  3. create a feature selection
  4. create a model validation
  5. predict response values of new data (if data is available)
  6. execute or store the created pipeline
  7. load and analyze the completed pipeline
