3D Scene

The Scene widget provides the three-dimensional visualisation of the loaded molecules.
All activated Representations are shown here.




The Scene has two distinct modes:
  1. Rotate Mode (Hotkey CTRL-R)
  2. Picking Mode (Hotkey CTRL-P)
The user can switch between the two modes, either by pressing on the according menu entries in the menu point "Display", or use the hotkeys.

Rotate Mode

In the Rotate Mode, the user can zoom, rotate or translate the view by pressing the mouse buttons and moving the mouse:

Button Effect
Left Rotate left/right and up/down
Middle Zoom in and out
Right Translate up/down and left/right


For user with less then three mouse buttons exist special combinations, so that BALLView works well also with one mouse button:
Mouse button + key pressed Effect
Left Rotate left/right and up/down
Left + Shift Zoom in and out
Left + Control Translate up/down and left/right

Picking Mode

The Picking Mode is designed to identify parts of molecules or mark them for special tasks. A special cursor occurs, an the user can either click on single items or moving the mouse button with pressed left or right mouse button to create a selection rectangle. Any picked objects become instantly colored (normaly yellow, see image on top). In the Molecular Control, the checkboxes for the entries of these items become checked. The user can also deselect picked objects by using the right button (or for users with only one mouse button, by using the left mouse button and the Shift-modifier).

Mouse button + key pressed Effect
Left Select geometric objects, e.g. for atoms
Right (or Left + Shift) Deselect geometric objects

Selection Color

The color for selected objects can be changed in the DisplayProperties dialog.

Background Color

The background color can be changed in the Preferences under the point "3D View"->"Common".
