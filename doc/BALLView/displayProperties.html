DisplayProperties



This is the main dialog for creating and modifiying Representations. It can be opened by either
  • the menu entry "Display->Display Properties"
  • the context menu in the MolecularControl
  • the context menu in the GeometricControl
The dialog can be used in two modi, depending on what items the user selecteded:
  • If a Representation in the GeometricControl is selected, the dialog will modifiy this Representation.
  • If a molecular entity in the MolecularControl is selected, the dialog will create a new Representation.
To distingish between these two modi, the caption of the dialog and the caption of the apply button in the lower left side will differ, depending on which mode is currently active.

Model Settings

Resolution

Every model can be created in at least three different resolutions to comply with vastly different graphic performances of todays computers. Try setting the detail level to "low" if the graphics are bucking while rotating the view. The detail level of all surfaces can be set in smaller steps by using the according slider below. Coloring Settings

Coloring Method

BALLView has ten different coloring methods available:
  • by element
  • by residue index
  • by residue name (a different color for every type of amino acid)
  • by secondary structure
  • by atom charge
  • by atom distance (use with Selection)
  • by temperature factor (read from PDB files)
  • by occupancy (read from PDB files)
  • by forces
  • custom (one single color)
Informations on how to color Representations from three-dimensional data grids can be found here.
